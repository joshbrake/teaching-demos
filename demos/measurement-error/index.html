Equations

Vrms = Vpp / (2√2)
Videal = Vrms · R2 / (R1 + R2)
Reff = R2 ‖ Zinst
Vloaded = Vrms · Reff / (R1 + Reff)
|Z| = R / √(1 + (2πfRC)²)
XC = 1 / (2πfC)

Key Insight

Loading is negligible when |Zinst| ≫ R2.

How It Works

A voltmeter has finite input impedance Zinst. When connected, it draws current and changes the voltage.

Real instruments have parallel capacitance (R ‖ C), making impedance decrease at higher frequencies. The capacitor provides a lower-impedance AC path.

This loading effect is worst when |Zinst| ≈ R2. A buffer (op-amp follower) eliminates loading by presenting very high input impedance.

Try: Switch between 200 Hz and 175 kHz to see how capacitance affects different instruments!

Definitions

Loading: Change in circuit behavior caused by the measurement device drawing current.
Vrms: Root-mean-square voltage. For a sine wave, Vrms = Vpp / 2√2.
R ‖ C: Parallel impedance. |Z| = R / √(1 + (2πfRC)²). Always ≤ R.
XC: Capacitive reactance = 1/(2πfC). When XC ≫ R, capacitance is negligible.
σV: Uncertainty in voltage from error propagation of component tolerances.
